Example of a game engine architecture using subsystems and systems:

    GameLoop
     ├─ InputSubsystem
     │    ├─ KeyboardInputSystem
     │    ├─ MouseInputSystem
     │    └─ GamepadInputSystem
     │
     ├─ AISubsystem
     │    ├─ DecisionMakingSystem
     │    ├─ PathfindingSystem
     │    ├─ BehaviorTreeSystem
     │    └─ Faction/RelationshipSystem
     │
     ├─ PhysicsSubsystem
     │    ├─ CollisionSystem
     │    ├─ RigidBodySystem
     │    ├─ CharacterControllerSystem
     │    └─ PhysicsTriggerSystem
     │
     ├─ MovementSubsystem
     │    ├─ PlayerMovementSystem
     │    ├─ NPCMovementSystem
     │    └─ VehicleMovementSystem
     │
     ├─ AnimationSubsystem
     │    ├─ SkeletalAnimationSystem
     │    ├─ BlendShapeSystem
     │    └─ StateMachineAnimationSystem
     │
     ├─ HealthSubsystem
     │    ├─ BuffSystem               // poison, regen, shields
     │    ├─ DamageSystem             // apply queued damage/heals
     │    └─ DeathSystem              // remove dead entities, trigger effects
     │
     ├─ CombatSubsystem
     │    ├─ MeleeAttackSystem
     │    ├─ RangedAttackSystem
     │    ├─ AbilitySystem
     │    └─ CooldownSystem
     │
     ├─ TickingSubsystem
     │    ├─ ScheduledTickSystem      // one-shot or delayed events
     │    ├─ PeriodicTickSystem       // repeated updates (blocks, timed effects)
     │    └─ RandomTickSystem         // random block or environment updates
     │
     ├─ InventorySubsystem
     │    ├─ PickupSystem
     │    ├─ DropSystem
     │    ├─ CraftingSystem
     │    └─ ItemDecaySystem
     │
     ├─ RenderingSubsystem
     │    ├─ MeshUpdateSystem
     │    ├─ MaterialSystem
     │    ├─ LightingSystem
     │    ├─ ParticleSystem
     │    └─ PostProcessingSystem
     │
     ├─ AudioSubsystem
     │    ├─ SpatialAudioSystem
     │    ├─ EventSoundSystem
     │    └─ MusicSystem
     │
     ├─ NetworkingSubsystem
     │    ├─ ClientPredictionSystem
     │    ├─ ServerReconciliationSystem
     │    ├─ EntitySyncSystem
     │    └─ RemoteEventSystem
     │
     ├─ WorldSubsystem
     │    ├─ ChunkLoadingSystem
     │    ├─ TerrainGenerationSystem
     │    ├─ DayNightCycleSystem
     │    └─ WeatherSystem
     │
     └─ UISubsystem
          ├─ HUDSystem
          ├─ InventoryUISystem
          ├─ TooltipSystem
          └─ NotificationSystem
